# Проект "Платформа для любителей силовых видов спорта с Ассистентом"

## О проекте

Это полнофункциональное веб-приложение, разработанное для атлетов и фитнес-энтузиастов. Платформа предоставляет
инструменты для ведения тренировочного дневника, анализа прогресса, а также получения персональных рекомендаций от
уникального ИИ-тренера.

Проект представляет собой монорепозиторий с двумя основными частями:

- **`backend`**: API на Node.js (Express) для обработки данных и бизнес-логики.
- **`frontend`**: Клиентское приложение на React, обеспечивающее пользовательский интерфейс.

## Технологический стек

| Категория    | Технология    | Описание                                                                          |
|:-------------|:--------------|:----------------------------------------------------------------------------------|
| **Frontend** | React         | Библиотека для построения пользовательских интерфейсов.                           |
|              | Redux         | Управление состоянием приложения, включая персистентность.                        |
|              | TypeScript    | Строгая типизация для повышения надежности кода.                                  |
|              | Vite          | Современная и быстрая система сборки.                                             |
|              | Tailwind CSS  | Утилитарный CSS-фреймворк для гибкой стилизации.                                  |
| **Backend**  | Node.js       | Среда выполнения JavaScript на сервере.                                           |
|              | Express       | Минималистичный фреймворк для создания API.                                       |
|              | MongoDB       | NoSQL база данных для хранения информации.                                        |
|              | JWT           | Реализация безопасной аутентификации на основе токенов.                           |
| **AI**       | OpenRoute API | Интеграция с моделью mistralai/mistral-7b-instruct-v0.2 для работы ИИ-ассистента. |

## Ключевые возможности

- **Аутентификация и роли:** Система регистрации и входа с разделением прав доступа (пользователь, модератор,
  администратор).
- **Дневник тренировок:** Возможность записи данных о выполненных упражнениях.
- **Анализ прогресса:** Визуализация динамики тренировочного тоннажа с помощью графиков.
- **ИИ-Ассистент:** Персональный тренер на базе OpenRoute API, который анализирует данные из дневника и дает
  рекомендации в уникальном стиле.
- **Управление контентом:** Администраторы могут создавать, редактировать и удалять статьи.
- **Интерактивность:** Пользователи могут оставлять комментарии к статьям.
- **Обратная связь:** Форма для сбора заявок от пользователей.

## Руководство по запуску

1. **Запустить Backend сервер:**

    ```bash
     cd backend
     npm start
    ```

Сервер будет доступен по адресу `http://localhost:3001`.

2. **Запустить Frontend приложение:**
   ```bash
   cd frontend
   npm run dev
   ```
   Приложение будет доступно по адресу `http://localhost:3000`.

-   **Избранное:** Добавление функционала для сохранения понравившихся статей и видео в личном кабинете.
-   **Расширенная аналитика:** Внедрение новых метрик и графиков для более детального анализа тренировочного процесса (например, по отдельным упражнениям).

-   **Сохранение истории чата с ассистентом между сессиями**
-   **Адаптивная версия под мобильные устройства**
